# NotebookLM-Estudo-AZ-900

Guia de estudos para a certificação **AZ-900: Fundamentos do Azure**

O projeto foi realizado neste link: [NotebookLM - AZ-900](https://notebooklm.google.com/notebook/9cfac469-53bf-4550-920d-e5ae0776d51f)

## Sobre o NotebookLM

O NotebookLM é uma ferramenta do Google baseada em inteligência artificial (IA) que funciona como um assistente de pesquisa, estudos e organização de informações. Auxilia os usuários a compreender documentos complexos, sintetizar informações e gerar novas ideias a partir de fontes específicas, de formatos variados, que o próprio usuário insere para evitar alucinações.

> **Importante:** Verificar sempre as referências citadas e garantir que os documentos são bem estruturados e sem informações contraditórias.

## Objetivo

Consolidar materiais e recursos de estudo para preparação da certificação AZ-900, utilizando fontes de vídeo, textos e práticas disponibilizadas pela Microsoft e comunidade.
